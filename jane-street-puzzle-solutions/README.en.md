## Repository structure

- `data/puzzles.json`: canonical archive manifest.
- `puzzles/<year>/...`: bilingual documents, metadata, code, tests, and research notes.
- `scripts/`: archive refresh, index generation, validation, and reporting tools.
- `templates/`: authoring templates.

## Installation and use

Use Python 3.11 or later. From this directory:

```bash
python3 -m pip install -r requirements.txt
python3 -m pytest
python3 scripts/validate_metadata.py
python3 scripts/validate_bilingual_content.py
python3 scripts/build_index.py
python3 scripts/check_links.py
python3 scripts/repository_report.py
```

Run a puzzle implementation from its own directory with `python3 solution.py`.
